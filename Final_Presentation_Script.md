# 🎤 최종 시연자를 위한 가이드북 (The Script)

이 문서는 프로젝트 내용을 모르는 팀원이라도 **그대로 따라 하기만 하면** DB의 핵심 기능을 완벽하게 보여줄 수 있도록 작성되었습니다.

---

## 0. 🖥️ 준비 사항
1.  **크롬 브라우저**를 엽니다.
2.  주소창에 `http://localhost:5174` (또는 5173) 입력하여 접속합니다.
3.  화면이 뜨면 준비 완료.

---

## 🎬 Scene 1: "제목을 몰라도 찾는다" (SKU 검색)
**목표**: DB가 텍스트 검색이 아닌, **표준 자산(SKU) 매핑**을 통해 검색한다는 것을 증명합니다.

1.  **[동작]**: 메인 화면 검색창 클릭.
2.  **[입력]**: `S24` 라고 입력하고 엔터.
3.  **[결과 확인]**:
    *   검색 결과 중 **"선물받은거 미개봉 급처합니다"**라는 상품이 나옵니다.
    *   **중요 포인트**: "제목에 'S24'라는 글자가 없는데도 검색되었습니다."
4.  **[설명/강조]**:
    *   마우스를 상품 카드 좌측 하단의 **`SKU: SM-S9210`** 뱃지에 올리세요.
    *   *"보시는 것처럼 제목은 제각각이지만, DB 내부에서는 표준 SKU 코드로 묶여 있기 때문에 정확한 검색이 가능합니다."*

---

## 🎬 Scene 2: "DB가 가격을 감시한다" (부정 거래 방지)
**목표**: 어플리케이션이 아닌 **DB 트리거**가 직접 데이터를 검증하고 차단하는 것을 보여줍니다.

1.  **[동작]**: 우측 상단 **[+ 등록하기]** 버튼 클릭.
2.  **[동작]**: **[티켓/교환권]** 카드 선택 -> **[다음]**.
3.  **[입력]**:
    *   **공연명**: `임영웅 콘서트` (대충 입력)
    *   **날짜/좌석**: 대충 아무거나 선택.
    *   **원가(Original Price)**: `100000` (10만원) 입력.
    *   **판매가(Sell Price)**: `200000` (20만원) 입력. (**200% 폭리**)
4.  **[동작]**: 하단 **[상품 등록 완료]** 버튼 클릭.
5.  **[결과 확인]**:
    *   🚨 **경고창 팝업**: "입력하신 판매가가 정가의 120%를 초과하여..."
6.  **[수정 및 성공]**:
    *   확인을 누르고, 판매가를 `110000` (11만원)으로 수정.
    *   다시 등록 클릭 -> **"성공"** 화면.
    *   *"백엔드 서버도 아닌, DB 엔진 레벨에서 Trigger가 악성 데이터를 원천 차단했습니다."*

---

## 🎬 Scene 3: "가격 거품을 시각화한다" (데이터 투명성)
**목표**: '원가'와 '판매가'를 비교하는 DB 로직이 화면에 어떻게 반영되는지 보여줍니다.

1.  **[동작]**: 메인으로 돌아가기.
2.  **[동작]**: 카테고리에서 **[티켓/교환권]** 클릭.
3.  **[결과 확인]**:
    *   상품 카드마다 **빨간색/초록색 뱃지**를 가리키세요. (`+15%`, `+5%`)
4.  **[설명/강조]**:
    *   *"저희 DB는 모든 티켓의 정가 데이터를 가지고 있습니다. 따라서 사용자는 일일이 계산할 필요 없이, 이 가격이 정가 대비 얼마나 비싼지 **프리미엄율(%)**로 바로 확인할 수 있습니다."*

---

## 🎬 Scene 4: "구조 없이 구조를 만든다" (JSONB 필터링)
**목표**: 스키마 변경 없이 JSON 데이터 내부를 쿼리하는 유연함을 보여줍니다.

1.  **[동작]**: 방금 들어온 티켓 리스트 페이지 상단을 보세요.
2.  **[동작]**: **[JSONB Tech Demo]** 라고 적힌 패널 확인.
3.  **[동작]**: **[VIP구역]** 버튼 클릭.
4.  **[결과 확인]**:
    *   화면이 깜빡이며 **[VIP석]**인 티켓들만 남습니다.
5.  **[설명/강조]**:
    *   *"이 '구역' 정보는 별도 컬럼이 아닙니다. 하나의 JSON 텍스트 덩어리 안에 들어있는데, 저희는 PostgreSQL의 JSON 인덱싱 기술로 이를 실시간으로 필터링했습니다."*
